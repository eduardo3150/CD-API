Manual básico de la API

Metodo Enlace Requiere Descripcion
GET http://174.138.64.78/CDAPI/ - Pagina principal, incluye el manual basico de la API
Artistas
GET http://174.138.64.78/CDAPI/artists - Devuelve un JSON con todos los Artistas/Bandas ingresados en la API
GET http://174.138.64.78/CDAPI/artist/[{id}] [{id}] = Identificador de un Artista en especifico Devuelve un JSON con la informacion sobre un artista/grupo en especifico por medio de su ID
GET http://174.138.64.78/CDAPI/artist/search/[{query}] [{query}] = Nombre de un artista Devuelve un JSON con la informacion sobre un artista/grupo en especifico por medio de su nombre
POST http://174.138.64.78/CDAPI/newartist
  • nombre : string
  • miembros : integer
  • trayectoria : integer
  • genero : string
  • primersencillo : string
Ingresa un nuevo registro de artista, devuelve una confirmacion cuando el ingreso se realiza con exito Ejemplo cuerpo JSON:
{ "nombre" : "The beatles", "miembros" : 4, "trayectoria": 10, "genero":"Rock", "primersencillo": "Love me do" }
DELETE http://174.138.64.78/CDAPI/artist/[{id}] [{id}] = Identificador de un Artista en especifico Elimina un elemento de artista por medio de su ID
PUT http://174.138.64.78/CDAPI/artist/[{id}] [{id}] = Identificador de un Artista en especifico
  • nombre : string
  • miembros : integer
  • trayectoria : integer
  • genero : string
  • primersencillo : string
Actualiza un registro existente de un artista, requiere que se ingrese en su url el identificador del artista a modificar. Ejemplo del cuerpo JSON:
{ "nombre" : "Aerosmith", "miembros" : 4, "trayectoria": 30, "genero":"Blues", "primersencillo": "Dream on" }
Proximos Conciertos
GET http://174.138.64.78/CDAPI/concerts - Devuelve un JSON con todos los proximos conciertos ingresados en la API
GET http://174.138.64.78/CDAPI/concert/[{id}] [{id}] = Identificador de un Concierto en especifico Devuelve un JSON con la informacion sobre un Concierto en especifico por medio de su ID
GET http://174.138.64.78/CDAPI/concert/search/[{query}] [{query}] = Nombre de un artista Devuelve un JSON con la informacion sobre todos los conciertos asociados a un determinado artista/grupo por medio de su nombre
POST http://174.138.64.78/CDAPI/newconcert
  • id_artista: integer
  • lugar: string
  • asistencia: integer
  • entrada: double
  • ganancias: double
  • pago_artista: double
  • fecha: date
  • representante: string
 Ingresa un nuevo registro de un concierto proximo, devuelve una confirmacion cuando el ingreso se realiza con exito, Es importante que se ingrese el identificador de un artista para que el registro se haga correctamente. Ejemplo cuerpo JSON:
{ "id_artista": 5, "lugar": "O2 Stadium", "asistencia": 50000, "entrada": 355.00, "ganancias": 105000, "pago_artista": 70000, "fecha": "2017-06-18", "representante": "Tyler Black" }
DELETE http://174.138.64.78/CDAPI/concert/[{id}] [{id}] = Identificador de un Concierto en especifico Elimina un elemento de proximo concierto por medio de su ID
PUT http://174.138.64.78/CDAPI/concert/[{id}] [{id}] = Identificador de un Concierto en especifico
  • id_artista: integer
  • lugar: string
  • asistencia: integer
  • entrada: double
  • ganancias: double
  • pago_artista: double
  • fecha: date
  • representante: string
 Actualiza un registro existente de un concierto, requiere que se ingrese en su url el identificador del concierto a modificar. Ejemplo del cuerpo JSON:
{ "id_artista": 5, "lugar": "Wembley Stadium", "asistencia": 100000, "entrada": 555.00, "ganancias": 1005000, "pago_artista": 700000, "fecha": "2017-06-18", "representante": "Chad Black" }
Discografia
GET http://174.138.64.78/CDAPI/discography - Devuelve un JSON con todos los proximos discos registrados en la API
GET http://174.138.64.78/CDAPI/disc/[{id}] [{id}] = Identificador de un Disco en especifico Devuelve un JSON con la informacion sobre un Disco en especifico por medio de su ID
GET http://174.138.64.78/CDAPI/disc/search/[{query}] [{query}] = Nombre de un artista Devuelve un JSON con la informacion sobre todos los Discos asociados a un determinado artista/grupo por medio de su nombre
POST http://174.138.64.78/CDAPI/newdisc
  • id_artista: integer
  • nombre: string
  • anio: int
  • ventas_copias: int
  • pistas: int
  • colaboracion: string
  • sencillo: string
 Ingresa un nuevo registro de un Disco, devuelve una confirmacion cuando el ingreso se realiza con exito, Es importante que se ingrese el identificador de un artista para que el registro se haga correctamente. Ejemplo cuerpo JSON:
{ "id_artista": 5, "nombre": "Rubber Soul", "anio": 1965, "ventas_copias": 100000, "pistas": 15, "colaboracion": "", "sencillo": "Drive my car" }
DELETE http://174.138.64.78/CDAPI/disc/[{id}] [{id}] = Identificador de un Disco en especifico Elimina un Disco por medio de su ID
PUT http://174.138.64.78/CDAPI/disc/[{id}] [{id}] = Identificador de un Disco en especifico
  • id_artista: integer
  • nombre: string
  • anio: int
  • ventas_copias: int
  • pistas: int
  • colaboracion: string
  • sencillo: string
 Actualiza un registro existente de un Disco, requiere que se ingrese en su url el identificador del Disco a modificar. Ejemplo del cuerpo JSON:
{ "id_artista": 5, "nombre": "Abbey Road", "anio": 1969, "ventas_copias": 1000000, "pistas": 14, "colaboracion": "none", "sencillo": "Come together" }
